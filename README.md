# World Store — E-commerce
World Store is a responsive and interactive e-commerce website for user and admin interface designed to preview products, place an orders, and user friendlywebsite. The application features a simple and attractive dashboard with linked data statistics, visual charts, and intuitive forms to add, edit, and delete order at order.html. It emphasizes a clear and attractive visualization design, compatible wit mobile and desktop, and ease of navigation, providing a best experience for user experience and user interface design for users and admins use.This page allows users to browse products, add them to a cart, view cart summary, and go through checkout with a modal form. It is built using static HTML, Tailwind CSS and vanilla JavaScript.

##  Features Included

- Product listing displayed in a responsive grid  
- Add products to a shopping cart and view cart items  
- Display total price in cart  
- Checkout modal with form fields: name, email, address, payment method (Credit Card / TNG E-wallet / FPX / Cash on Delivery)  
- Conditional form fields (e.g. credit-card fields hidden until “Credit Card” selected)  
- Responsive design — works on desktop and mobile  
- Clean UI using Tailwind CSS + SVG icons  

##  Instructions to Test / Use Page (Login)

This project include authentication and  login functionality.  
To test the page:  
1. Open the live website link, or open `index.html` (or the main HTML file) locally in a browser.
2. Enter username and password  / guest (demo)  
3. Browse products, add items to cart, and click **Checkout**.  
4. Fill in the checkout form and submit.  
5. Checkout process (payment logic) is simulated in JavaScript — no real payment or backend required.  

>  Note: Since there is no backend or database, all data is hardcoded in JavaScript arrays, and changes reset on page reload.
